Quickstart¶

PHARO reduction¶

We’ll walk through an example of using this code on the PHARO data contained within the test suite.

Note

The aodirs.py file and associated references will be deprecated immediately after the open-source release of this code.

First, make sure that you edit the aodirs.py file. We’d like to set the raw_dir function to the directory containing src/simmer/tests/PHARO_integration. Set the reddir function to return whatever directory you’d like your reduced data to be dumped in. Make sure that the config function returns the path to the config file in the shane-reduction directory.

Next, run the below line instantiating the code:

import insts as i
import drivers

inst = i.PHARO()

This instantiates inst as an instrument object prepared to reduced PHARO data.

Next, run

drivers.all_driver(inst)

Progress bars should give estimates as to how long each step will take along the way.

Now, check your reddir folder — you should find your reduced data!

A reduction with ShARCS data would work similarly — with appropriate adjustments to raw_dir, reddir, config, and inst (to i.ShARCS).
