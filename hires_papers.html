High-resolution spectroscopy literature database

Aggregating all the literature on high-resolution spectroscopy of exoplanet atmospheres in one spot.

There's been a rapid increase in the number of high-resolution spectroscopy studies in recent years. With all the science going on simultaneously, it's hard to keep track of what's already been done.

Status

Along with collaborators Hayley Beltz and Isaac Malsky, we present a mostly complete collection of all high-resolution spectroscopy research. We've also automated some ArXiv-scraping from here on out, but manual checks for integration are still required.

This is an ongoing project, and we're always looking for input! Check out our GitHub Project for the status of this work. If you find this resource useful in your own work, please acknowledge our Zenodo listing.
